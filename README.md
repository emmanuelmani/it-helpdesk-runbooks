# IT Helpdesk Runbooks

A personal collection of IT support runbooks and reference guides I use and have built over four years of IT support work. These cover the most common scenarios that come up in a helpdesk role — onboarding, offboarding, device setup, and troubleshooting.

I built these because the fastest way to reduce repeat support tickets is good documentation. At DXC Technology I maintained a knowledge base that reduced repeat incidents by 30% across the team. These are the kinds of articles that made that happen.

## How to use these

Each document is a standalone guide. Copy, adapt, and use them. If you are an IT person setting up runbooks for your team, these are a starting point — every environment is different, so adjust the specific tool names to match what your organisation uses.
